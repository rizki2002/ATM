Petunjuk Penggunaan

1. Jalankan Apps dengan dari file atm_banking.java
2. Masuk Login melalui PIN ATM
3. untuk alice memiliki PIN atm : 12345
   untuk bob memiliki PIN atm : 2345
4. Masuk dengan memilih menu (sesuaikan dengan angka menu tersebut)
5. Untuk transfer masukan nama dengan orang yang dituju (alice atau bob)
6. jika sudah logout maka sistem akan keluar sehingga dibutuhkan running kembali
